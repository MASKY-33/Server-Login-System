# Server Login System
This system allows a user to enter a username and password, and verifies the login against a stored user database.
It simulates a simple backend‑style authentication flow using Python dictionaries and functions.

## Features
- Stores usernames and passwords in a dictionary
- Checks if the entered username exists
- Verifies if the password matches the stored password

- Returns clear messages:
- “Login successful”
- “Password incorrect”
- “User unknown”

- Uses a clean function (verify_login) for authentication logic

# How to use
Run the system and enter your username and password.
éThe program will validate your credentials and print the result.


## Learning Purpose
This project was created as part of backend engineering practice.
The goal was to improve skills in:
- dictionaries as mini‑databases
- authentication logic
- functions and return values
- input handling
- system flow and architecture thinking
